## サイトについて
Next.js, React, Tailwindを使用してサイトを作成しました。

## 開発環境
<ul>
<li>Ubuntu 22.04LTS</li>
<li>TypeScript 4.9.5</li>
<li>Node.js v18.14.1</li>
<li>Next.js 13.2.3</li>
<li>Tailwind 3.2.7</li>
</ul>

## スタート方法
モジュールのインストール
<code>
npm install
</code>

デバックで起動
<code>
npm run dev
</code>

ビルドして起動
<code>
npm run build
npm run start
</code>
